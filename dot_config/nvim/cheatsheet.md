# Neovim Cheat Sheet
F7        → New empty buffer  
F8        → Previous buffer  
F9        → Next buffer  
F10       → Close buffer (smart)
---
gd        → Go to definition  
gD        → Go to declaration  
gi        → Go to implementation  
gr        → Find references  
K         → zeigt Docstring / Type Info
---
Ctrl + o  → Jump back  
Ctrl + i  → Jump forward  
---
<leader>ds → Document symbols (current file)  
<leader>ws → Workspace symbols (project-wide)
Leader = Space
---
gcc       → Toggle comment current line  
gc + selection → Comment marked lines
---
ä         → Copy whole file to system clipboard  
---
ciw → Change inner word
diw → Delete inner word
yiw → Yank inner word
ci" → Change inside quotes
ci( → Change inside parentheses
ci{ → Change inside braces
---
>ap → indent paragraph (Block)
<ap → unindent paragraph
---
[[ → vorherige Funktion/Klasse
]] → nächste Funktion/Klasse
---
fX → springe vorwärts zum nächsten X
tX → springe bis vor X
FX → rückwärts
